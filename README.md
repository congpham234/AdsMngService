### How to run locally:

Step 1: Build the project
```
npm run build
``` 

Step 2: Then start the server with

```
npm run start
```

The server should be served at port 3000 by default


--------------------
### How to run on AWS lambda:
Step 1: Build the project
```
npm run build
```

Step 2: Import the `index.zip` file in `dist` folder to AWS lambda
